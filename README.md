# cb-loc-count
  
`cb-loc-count` is a command-line tool for counting lines of code in a SonarQube instance.

## Usage

```
export SONAR_URL="YOUR_SONARQUBE_URL"
export SONAR_TOKEN="YOUR_SONAR_TOKEN"
./cb-loc-count.sh
```
